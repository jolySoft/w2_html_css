Coding

Python

Python 3 was my first language, my first Hello World. I learned the basics of Python 3 following the CodeCademy course which I ended up doing twice (not that I am proud of... but I love normalising the learning process and that it is OK to not understand or pick things up from the first take). After finishing my second round of the course I wanted to try my first mini projects outside of the platform so I could install Python, an IDLE (I chose PyCharm) and get my head around a more realistic coding environement.

One of the most challenging things I faced was to come up with an idea. Once I had the idea or something that really interested me, it was fun to see how I could get stuck, search the internet (mainly stackoverflow) for answers and then get like a sugar rush when things worked.

I know they are easy projects... but I still like the time spent while doing them and I am proud of what I accomplished being a complete beginner to code!

Hangman game

Very beginner friendly game, I created a hangman with the Schengen countries. Although I know it is a simple project, it was s good first one to do mainly because:

  • It is doable for a pure beginner with a base of Python, the fact of facing something achievable was very positive for me so I wouldn’t feel demotivated on my first project thinking I wasn’t cut for this!
  • Practice a lot of if, elif, else including loops and knowing when to break.
  • It allowed me to start playing around with functions creating one for the hangman display and also for game over so I could easily call those every time I needed them.
  • Use of the random module, in my case I used the random.choice() for the selection of the word to be guessed.
  • It’s a game…so it’s fun to test once it’s finished! Also, as it is an easy game, it is very easy to test yourself different scenarios: guessing the word before all my tries are consumed, not guessing at all the word, what if I input a number instead of a letter, etc?

Netflix analysis

One day, while speaking with my mum about how the pandemic increased our time spent watching Netflix, I said that I would love to compare the data to see who had the biggest increase in time on Netflixby comparing before and during the pandemic. I must confess, my mum was terrified of the idea (as she knew how much she was binge watching!) which made it even funnier for me to get into it!

First step was to get the Netflix data from our account. As we have a family account, I could have the data for the whole account . After a few days, I received the notification about the data being now available so I downloaded it and wow… there was a lot of data! Once my first shock had passed, I saw that I couldn’t just start the process straight as I imagined, so I decided to take my time to:

    Familiarise myself with all the data gathered so I could identify the documents and csv that contained the information to be imported for the project. Structure my project: what do I really want from it?

Then I was ready, I decided to structure my project in 2 parts:

  • Analytical part: this will fully dedicated to import the data needed and then create the different functions to extract information such as the shows watched, compare data, time watched, calculation of increases between period.
  • User part: I wanted to show the user the following options where the user could select what they wanted to view and the result would be prompted.

Again, I know it looks simple but I found some challenges while trying to code which I enjoyed solving and I was able to make it work as I wanted. Again, with some shy feeling from my part but wanting to be more open with my coding journey, here is my code in case you want to check out what I did.

Little spoiler alert here… my mum doubled my increase in hours watched, that’s why she got nervous when I started the project !

MySQL

CFG Data & SQL course

I enrolled on an intensive 4 weeks course with 2 sessions per week which was remotely taught via Zoom and using Slack to interact and collaborate between course students and instructors.

The content of the MySQL course included:

  • Database creation
  • Table creation and data insertion
  • Relational databases
  • Aggregation, order, group by, having
  • Joins and subqueries
  • Stored functions
  • Stored procedures
  • Triggers
  • Views

Something that amazed me the most from the course was the feeling of camaraderie we all had seeing students helping each other… I loved it! We had like a “safe space” to post out queries, where we struggled and even share out code so other could help us out. Even if I couldn’t help out, I liked to read others code and responses and how they solved the problem as it helped me learn more.

PlanShop

My project was a database to store recipes with its ingredients so I could automatically create my shopping list by selecting the recipes I wanted to include in my next week’s meal plan. In addition I added the possibility to have different supermarkets so I could also compare where would it be cheaper to buy the products from

One of the things I struggled the most with my project was the initial part of building my database and tables as I wanted my data to be as normalised as possible. I started designing something in Excel so I could get an idea of the tables and type or variables I would have in each of them including how they would be linking to each other… but then as I went into practice some things were not as polished as I thought they were which resulted in me having to drop tables (and the whole database) a few times until I was fully happy with it. Even if this part felt a little frustrating I must recognise that I learned a lot from it and I felt a great satisfaction when I finally got it.

Once the project was completed… it was time to present it to my colleagues and instructors. I was super nervous, it was my first time talking to someone about something code/technical related… but I did it! As I look back to it, I would love to improve a few things on my presentation but when I take into consideration the time I had available and the learning curve I went through, I am proud of myself… here I was, presenting a mini project on something I learned in only 4 weeks.
